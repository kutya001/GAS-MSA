# Product Context — МобилТрек Pro

> **Дата последнего обновления:** 2026-04-09

## Какую бизнес-задачу решает программа

**МобилТрек Pro** — система учёта для бизнеса по продаже мобильных устройств. Решает следующие задачи:

### Основные боли бизнеса
1. **Учёт закупок** — отслеживание устройств от поставщика до склада с фиксацией стоимости в USD и пересчётом в KGS по курсу.
2. **Складской учёт** — мгновенное понимание: сколько товара на каждом складе, на какую сумму.
3. **Продажи и долги** — отслеживание продаж, рассрочек, остатков долга покупателей.
4. **Кассовый учёт** — контроль денежных потоков по кошелькам (мульти-валюта).
5. **Номенклатура** — гибкое управление характеристиками товаров через EAV-модель (MDM), позволяющее добавлять новые свойства без изменения схемы БД.
6. **Аналитика** — KPI дашборд: выручка, прибыль, тренды, долги, залежавшийся товар.
7. **Публичный каталог** — витрина PhoneMarket (`?p=catalog`) для публикации всего товарного ассортимента с актуальным наличием со склада.
8. **Маркетплейс** — standalone каталог (`?p=marketplace`) с продавцами, ценами (новые/б/у), отзывами, WhatsApp-связью с продавцом.

### Целевая аудитория
- Владельцы/менеджеры точек продаж мобильных устройств в Кыргызстане.
- Малый бизнес, который не может позволить себе дорогие ERP-системы.

### Технические ограничения
- Работает полностью на Google Apps Script + Google Sheets как база данных.
- Бесплатная инфраструктура (Google Workspace).
- Лимит выполнения: 6 минут на вызов.
- Развёртывание через `clasp` (CLI для GAS).

### Конкурентные преимущества
- **Нулевая стоимость** инфраструктуры.
- **Мгновенный деплой** через GAS Web App.
- **MDM-конструктор** — позволяет создавать произвольные атрибуты товаров.
- **Materialized-колонки** — быстрые чтения баланса/остатков без пересчёта.
- **Публичная витрина** — каталог товаров с наличием для клиентов/партнёров, без отдельного хостинга.
- **Маркетплейс** — мульти-продавец каталог с ценами (новые/б/у), отзывами, WhatsApp-контактами, тёмной темой — без отдельного хостинга.
